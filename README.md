# Job Search Website

Современный дизайн платформы для поиска работы и вакансий. Макет создан в Figma и предназначен для удобного поиска подходящих вакансий, просмотра компаний и отправки заявок.

## О проекте

Данный проект представляет собой сайт для поиска работы, где пользователи могут находить вакансии, изучать информацию о компаниях и откликаться на предложения работодателей.

## Секции сайта

### Главный экран
Первый экран с основным заголовком, поисковой строкой и кнопкой для быстрого поиска вакансий.

### Categories
Секция с популярными категориями профессий и направлений работы.

### Featured Jobs
Блок с рекомендуемыми или популярными вакансиями.

### Companies
Секция с информацией о компаниях и работодателях.

### About Platform
Информационный блок о возможностях платформы и преимуществах для пользователей.

### Testimonials
Отзывы пользователей о поиске работы через платформу.

### Statistics
Секция со статистикой: количество вакансий, компаний и пользователей.

### CTA Section
Блок с призывом зарегистрироваться или начать поиск работы.

### Footer
Нижняя часть сайта с навигацией, контактной информацией и социальными сетями.
